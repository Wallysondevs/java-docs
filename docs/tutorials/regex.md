# Expressões Regulares

Esta seção ensina a sintaxe de expressões regulares suportada pela API `java.util.regex` e apresenta vários exemplos práticos para ilustrar como os diversos objetos interagem.

1.  [Introdução às Expressões Regulares](<#/doc/tutorials/regex/intro>)  
    Introdução às expressões regulares: o que são e o que você pode fazer com elas. Inclui o código usado para testar expressões regulares, utilizado ao longo desta seção. 

2.  [Literais de String](<#/doc/tutorials/regex/string-literals>)  
    Apresenta correspondência de padrões básica, metacaracteres e escape.

3.  [Classes de Caracteres](<#/doc/tutorials/regex/character-classes>)  
    Descreve classes de caracteres simples, negação, intervalos, uniões, interseções e subtração.

4.  [Classes de Caracteres Predefinidas](<#/doc/tutorials/regex/predefined-character-classes>)  
    Descreve as classes de caracteres predefinidas básicas para caracteres de espaço em branco, palavra e dígito.

5.  [Quantificadores](<#/doc/tutorials/regex/quantifiers>)  
    Explica quantificadores gulosos (greedy), relutantes (reluctant) e possessivos (possessive) para corresponder a uma expressão especificada x número de vezes.

6.  [Grupos de Captura](<#/doc/tutorials/regex/groups>)  
    Explica como tratar múltiplos caracteres como uma única unidade.

7.  [Limites](<#/doc/tutorials/regex/boundary-matchers>)  
    Descreve limites de linha, palavra e entrada.

8.  [A Classe Pattern](<#/doc/tutorials/regex/patterns>)  
    Examina outros métodos úteis da classe Pattern e explora recursos avançados como compilação com flags e uso de expressões de flag embutidas.

9.  [A Classe Matcher](<#/doc/tutorials/regex/matchers>)  
    Descreve os métodos comumente usados da classe Matcher.

10. [Métodos da Classe PatternSyntaxException](<#/doc/tutorials/regex/pattern-syntax-exception>)  
    Descreve como examinar uma PatternSyntaxException.
