# Genéricos

1.  [Introdução aos Genéricos](<#/doc/tutorials/generics/intro>)

    Genéricos permitem que tipos (classes e interfaces) sejam parâmetros ao definir classes, interfaces e métodos. Assim como os parâmetros formais mais familiares usados em declarações de método, os parâmetros de tipo fornecem uma maneira de reutilizar o mesmo código com diferentes entradas.

2.  [Inferência de Tipo](<#/doc/tutorials/generics/type-inference>)

    Inferência de tipo é a capacidade de um compilador Java de analisar cada invocação de método e a declaração correspondente para determinar o argumento de tipo (ou argumentos) que tornam a invocação aplicável.

3.  [Curingas](<#/doc/tutorials/generics/wildcards>)

    Em código genérico, o ponto de interrogação (?), chamado de curinga, representa um tipo desconhecido. A seção a seguir discute os curingas em mais detalhes, incluindo curingas com limite superior, curingas com limite inferior e captura de curinga.

4.  [Apagamento de Tipo](<#/doc/tutorials/generics/type-erasure>)

    Apagamento de tipo garante que nenhuma nova classe seja criada para tipos parametrizados; consequentemente, os genéricos não incorrem em sobrecarga de tempo de execução.

    Restrições no uso de Genéricos.
