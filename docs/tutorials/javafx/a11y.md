# Acessibilidade em JavaFX

A Acessibilidade (a11y) garante que as aplicações possam ser percebidas, operadas e compreendidas por pessoas que utilizam tecnologias assistivas como leitores de tela, lupas e dispositivos de entrada alternativos.

JavaFX fornece APIs e propriedades de acessibilidade que o ajudam a construir interfaces de usuário inclusivas. Um bom ponto de partida é confiar nos controles padrão e garantir que sua interface de usuário empregue a semântica adequada. A partir daí, você pode personalizar metadados acessíveis e ajustar como os componentes se comportam para tecnologias assistivas.

O uso avançado das APIs de Acessibilidade do JavaFX inclui a implementação de controles totalmente personalizados que expõem funções, estados, nomes, descrições e ações corretos, além de validar o comportamento com tecnologias assistivas e testes automatizados de interface de usuário em todas as plataformas.
