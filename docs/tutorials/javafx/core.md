# Fundamentos do JavaFX

Esta página foi contribuída por [Gail C. Anderson](</author/GailC.Anderson>) e [Paul Anderson](</author/PaulAnderson>) sob a [UPL](<https://oss.oracle.com/licenses/upl/>) e é de [The Definitive Guide to Modern Java Clients with JavaFX 17](<https://link.springer.com/book/10.1007/978-1-4842-7268-8>) gentilmente cedido pela Apress.

Com o Java SDK e o JavaFX instalados em seu sistema, vamos criar algumas aplicações e explorar os fundamentos do JavaFX. Primeiro, descreveremos a estrutura básica de uma aplicação JavaFX, juntamente com recursos selecionados que tornam o JavaFX uma escolha poderosa para clientes modernos. Mostraremos como criar UIs que são atraentes e responsivas. Abordaremos o FXML, a linguagem de marcação baseada em XML que permite definir e configurar sua UI. Também apresentaremos o Scene Builder, um utilitário autônomo de arrastar e soltar para projetar e configurar uma UI JavaFX.

Para refinar ainda mais ou reestilizar completamente sua UI, o JavaFX usa Cascading Style Sheets (CSS). Mostraremos várias maneiras de usar CSS com JavaFX.

As properties do JavaFX fornecem um poderoso mecanismo de binding. Apresentaremos as properties e o binding do JavaFX. Mostraremos por que os observables e o binding do JavaFX ajudam a criar código compacto que é menos propenso a erros do que listeners volumosos. Também exploraremos vários controles de layout e mostraremos como é fácil incorporar animação em sua UI.

Concluiremos a série com uma aplicação de exemplo que implementa uma UI mestre-detalhe usando collections do JavaFX, um formulário editável e botões para operações CRUD típicas de banco de dados. Isso, então, tem como objetivo dar a você uma amostra do que é possível com o JavaFX e fornecer os fundamentos para explorar o JavaFX ainda mais ao longo deste livro. Vamos começar!
