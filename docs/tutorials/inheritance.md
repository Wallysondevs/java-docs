# Herança

A programação orientada a objetos permite que classes herdem estado e comportamento comumente usados de outras classes. Esta seção aborda como a herança é tratada pela linguagem Java.

1.  [Herança](<#/doc/tutorials/inheritance/what-is-inheritance>)

    Criando classes que derivam de outras classes, herdando campos e métodos dessas classes.

2.  [Sobrescrevendo e Ocultando Métodos](<#/doc/tutorials/inheritance/overriding>)

    Sobrescrevendo métodos de instância em uma subclasse.

3.  [Polimorfismo](<#/doc/tutorials/inheritance/polymorphism>)

    Escolhendo o método correto para executar em tempo de execução.

4.  [Object como uma Superclasse](<#/doc/tutorials/inheritance/objects>)

    Listando os métodos da classe Object.

5.  [Métodos e Classes Abstratas](<#/doc/tutorials/inheritance/abstract-classes>)

    Compartilhando métodos comuns entre classes com classes abstratas.
