# Ferramentas JDK Complementares

Além das ferramentas distribuídas com cada lançamento do JDK, há uma série de ferramentas enviadas separadamente que são igualmente úteis ao desenvolver, testar, monitorar ou perfilar sua aplicação. Algumas dessas ferramentas fazem parte do [Code Tools Project](<https://openjdk.org/projects/code-tools/>), e elas ajudam principalmente desenvolvedores que trabalham na base de código do OpenJDK. No entanto, ferramentas como `jextract` provaram ser valiosas além do uso do OpenJDK, pois analisam arquivos de cabeçalho de bibliotecas nativas e geram código Java, simplificando assim a integração com uma variedade de bibliotecas nativas.

1. [Jextract - A Ferramenta de Extração de Bindings de Biblioteca Nativa](<#/doc/tutorials/jvm/tools/complementary/jextract>)

Execute jextract e use os bindings Java gerados para integrar bibliotecas nativas em sua aplicação.
