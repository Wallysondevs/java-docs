# Expressões Lambda

As expressões lambda foram uma adição poderosa à linguagem Java a partir do Java 8. Esta é uma série de tutoriais com o objetivo de introduzir o conceito de lambdas enquanto ensina incrementalmente como usá-las na prática à medida que você avança em cada tutorial.

Os tutoriais desta série estão listados abaixo. Recomendamos começar pelo primeiro e seguir em frente, mas você está livre para começar onde quiser!

1. [Escrevendo Sua Primeira Expressão Lambda](<#/doc/tutorials/lambdas/first-lambdas>)
Escrevendo sua primeira lambda, encontrando o tipo de uma lambda.

2. [Usando Expressões Lambda em Sua Aplicação](<#/doc/tutorials/lambdas/functional-interfaces>)
Descobrindo as interfaces funcionais mais úteis do JDK.

3. [Escrevendo Expressões Lambda como Method References](<#/doc/tutorials/lambdas/method-references>)
Compreendendo Method References.

4. [Combinando Expressões Lambda](<#/doc/tutorials/lambdas/combining-chaining-composing>)
Usando Default e Static Methods para Combinar e Criar Lambdas.

5. [Escrevendo e Combinando Comparators](<#/doc/tutorials/lambdas/writing-comparators>)
Criando e Combinando Comparators Usando Expressões Lambda.
