# A API Java I/O

Java I/O é um conjunto de classes que fornecem acesso a recursos externos, incluindo sistemas de arquivos e a rede.

Este tutorial irá guiá-lo através da API Java I/O, apresentando as noções básicas que você precisa entender para começar a escrever código Java que aproveite a API.

1. [Compreendendo os Principais Conceitos de Java I/O](<#/doc/tutorials/java-io/intro>)

Introduzindo a API Java I/O: arquivos, caminhos, streams de caracteres e streams de bytes.

2. [Noções Básicas do Sistema de Arquivos](<#/doc/tutorials/java-io/file-system>)

Trabalhando com o sistema de arquivos: trabalhando com arquivos e caminhos, trabalhando com diretórios, obtendo metadados de arquivos.

3. [Noções Básicas de Operações de Arquivo](<#/doc/tutorials/java-io/reading-writing>)

Escrevendo e lendo arquivos: conhecendo a diferença entre arquivos de texto e arquivos binários.

4. [Juntando Tudo](<#/doc/tutorials/java-io/putting-it-all-together>)

Um caso de uso complexo para utilizar todas as estruturas e decorações de Java I/O.
