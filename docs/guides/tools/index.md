# Especificações das Ferramentas do Java® Development Kit Versão 25

## Todas as Plataformas

  * [jar](<#/doc/guides/tools/jar>) - cria um arquivo para classes e recursos, e manipula ou restaura classes ou recursos individuais de um arquivo
  * [jarsigner](<#/doc/guides/tools/jarsigner>) - assina e verifica arquivos Java Archive (JAR)
  * [java](<#/doc/guides/tools/java>) - inicia uma aplicação Java
  * [javac](<#/doc/guides/tools/javac>) - lê definições de classes e interfaces Java e as compila em bytecode e arquivos de classe
  * [javadoc](<#/doc/guides/tools/javadoc>) - gera páginas HTML de documentação de API a partir de arquivos fonte Java
  * [javap](<#/doc/guides/tools/javap>) - desmonta um ou mais arquivos de classe
  * [jcmd](<#/doc/guides/tools/jcmd>) - envia requisições de comandos de diagnóstico para uma Java Virtual Machine (JVM) em execução
  * [jconsole](<#/doc/guides/tools/jconsole>) - inicia um console gráfico para monitorar e gerenciar aplicações Java
  * [jdb](<#/doc/guides/tools/jdb>) - encontra e corrige bugs em programas da plataforma Java
  * [jdeprscan](<#/doc/guides/tools/jdeprscan>) - ferramenta de análise estática que escaneia um arquivo jar (ou alguma outra agregação de arquivos de classe) em busca de usos de elementos de API obsoletos
  * [jdeps](<#/doc/guides/tools/jdeps>) - inicia o analisador de dependência de classes Java
  * [jfr](<#/doc/guides/tools/jfr>) - analisa e imprime arquivos Flight Recorder
  * [jhsdb](<#/doc/guides/tools/jhsdb>) - anexa a um processo Java ou inicia um depurador post-mortem para analisar o conteúdo de um core dump de uma Java Virtual Machine (JVM) que falhou
  * [jinfo](<#/doc/guides/tools/jinfo>) - gera informações de configuração Java para um processo Java especificado
  * [jlink](<#/doc/guides/tools/jlink>) - monta e otimiza um conjunto de módulos e suas dependências em uma imagem de tempo de execução personalizada
  * [jmap](<#/doc/guides/tools/jmap>) - imprime detalhes de um processo especificado
  * [jmod](<#/doc/guides/tools/jmod>) - cria arquivos JMOD e lista o conteúdo de arquivos JMOD existentes
  * [jnativescan](<#/doc/guides/tools/jnativescan>) - ferramenta de análise estática que escaneia um ou mais arquivos jar em busca de usos de funcionalidades nativas
  * [jpackage](<#/doc/guides/tools/jpackage>) - empacota uma aplicação Java autocontida
  * [jps](<#/doc/guides/tools/jps>) - lista as JVMs instrumentadas no sistema de destino
  * [jrunscript](<#/doc/guides/tools/jrunscript>) - executa um shell de script de linha de comando que suporta modos interativo e em lote
  * [jshell](<#/doc/guides/tools/jshell>) - avalia interativamente declarações, instruções e expressões da linguagem de programação Java em um loop de leitura-avaliação-impressão (REPL)
  * [jstack](<#/doc/guides/tools/jstack>) - imprime rastreamentos de pilha Java de threads Java para um processo Java especificado
  * [jstat](<#/doc/guides/tools/jstat>) - monitora estatísticas da JVM
  * [jstatd](<#/doc/guides/tools/jstatd>) - monitora a criação e o encerramento de VMs Java HotSpot instrumentadas
  * [jwebserver](<#/doc/guides/tools/jwebserver>) - inicia o Servidor Web Simples Java
  * [keytool](<#/doc/guides/tools/keytool>) - gerencia um keystore (banco de dados) de chaves criptográficas, cadeias de certificados X.509 e certificados confiáveis
  * [rmiregistry](<#/doc/guides/tools/rmiregistry>) - cria e inicia um registro de objeto remoto na porta especificada no host atual
  * [serialver](<#/doc/guides/tools/serialver>) - retorna o `serialVersionUID` para uma ou mais classes em um formato adequado para ser copiado para uma classe em evolução

## Somente Windows

  * [jabswitch](<#/doc/guides/tools/jabswitch>) - habilita ou desabilita o Java Access Bridge
  * [jaccessinspector](<#/doc/guides/tools/jaccessinspector>) - examina informações acessíveis sobre os objetos na Java Virtual Machine usando a API Java Accessibility Utilities
  * [jaccesswalker](<#/doc/guides/tools/jaccesswalker>) - navega pelas árvores de componentes em uma Java Virtual Machine específica e apresenta a hierarquia em uma visualização em árvore
  * [javaw](<#/doc/guides/tools/java>) - inicia uma aplicação Java sem uma janela de console
  * [kinit](<#/doc/guides/tools/kinit>) - obtém e armazena em cache tickets de concessão de tickets Kerberos
  * [klist](<#/doc/guides/tools/klist>) - exibe as entradas no cache de credenciais local e na tabela de chaves
  * [ktab](<#/doc/guides/tools/ktab>) - gerencia os nomes principais e as chaves de serviço armazenados em uma tabela de chaves local
