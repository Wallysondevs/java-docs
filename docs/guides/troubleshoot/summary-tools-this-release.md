# Resumo das Ferramentas Nesta Versão

## E Resumo das Ferramentas Nesta Versão

Este apêndice fornece um resumo das ferramentas disponíveis na versão atual do JDK.

Todas as ferramentas de solução de problemas do JDK descritas neste documento estão disponíveis no Linux.

As seguintes ferramentas de solução de problemas do JDK também estão disponíveis no Windows:

  * Flight Recorder

  * `jcmd`

  * JConsole

  * Java Virtual Machine

  * `jdb`

  * `jinfo`

  * `jmap`

  * `jps`

  * `jrunscript`

  * `jstack`

  * `jstat`

  * `jstatd`

  * `visualgc`
