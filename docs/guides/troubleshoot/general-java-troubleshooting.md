# Solução de Problemas Gerais do Java

## Parte I Solução de Problemas Gerais do Java

Esta parte descreve técnicas gerais de solução de problemas do Java e contém os seguintes tópicos.

  * [Prepare o Java para Solução de Problemas](<#/doc/guides/troubleshoot/prepare-java-troubleshooting>)

Fornece diretrizes para configurar tanto o Java quanto uma aplicação Java para melhores técnicas de solução de problemas. Essas configurações proativas do Java ajudam a depurar e a restringir problemas com o Java e uma aplicação Java.

  * [Ferramentas de Diagnóstico](<#/doc/guides/troubleshoot/diagnostic-tools>)

Descreve várias ferramentas de diagnóstico e monitoramento usadas com o Java Development Kit (JDK). Descreve ainda as ferramentas de solução de problemas disponíveis e explica o desenvolvimento de ferramentas personalizadas usando interfaces de programação de aplicação (APIs).

  * [Solucionar Problemas de Vazamentos de Memória](<#/doc/guides/troubleshoot/troubleshooting-memory-leaks>)

Fornece sugestões para diagnosticar problemas envolvendo possíveis vazamentos de memória.

  * [Solucionar Problemas de Desempenho Usando o Flight Recorder](<#/doc/guides/troubleshoot/troubleshoot-performance-issues-using-jfr>)

Identifica problemas de desempenho com uma aplicação Java e depura problemas usando o Java Flight Recorder.

  * [Solucionar Problemas de APIs de Segurança](<#/doc/guides/troubleshoot/troubleshoot-security-apis>)

Fornece um link para depurar problemas de segurança do Java.
