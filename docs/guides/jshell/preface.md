# Prefácio

## Prefácio

Este guia fornece informações sobre como usar a ferramenta Java Shell (JShell) para explorar a linguagem Java e prototipar código.

### Público

Este documento é destinado a desenvolvedores interessados em usar uma ferramenta Read-Eval-Print Loop (REPL) para aprender a linguagem Java, explorar novas APIs e recursos Java, e prototipar código complexo.

### Convenções

As seguintes convenções são usadas neste documento:

Convenção | Significado
---|---
negrito | O tipo negrito indica elementos da interface gráfica do usuário associados a uma ação, ou termos definidos no texto ou no glossário.
itálico | O tipo itálico indica títulos de livros, ênfase ou variáveis de espaço reservado para as quais você fornece valores específicos.
`monospace` | O tipo monospace indica comandos dentro de um parágrafo, URLs, código em exemplos, texto que aparece na tela ou texto que você insere.
`<keys>` | O nome de uma tecla do teclado ou combinação de teclas entre colchetes angulares (<>) indica que a tecla ou combinação de teclas é pressionada como parte da entrada do usuário.
