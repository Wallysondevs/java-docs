# Java Management Extensions (JMX) API

## 8 Java Management Extensions (JMX) API

A Java Management Extensions (JMX) API é uma API padrão para gerenciamento e monitoramento.

Os seguintes pacotes de documentação gerados pela ferramenta JavaDoc são fornecidos para a JMX API:

  * [javax.management](<https://docs.oracle.com/en/java/javase/25/docs/api/java.management/javax/management/package-summary.html>)

  * [javax.management.modelmbean](<https://docs.oracle.com/en/java/javase/25/docs/api/java.management/javax/management/modelmbean/package-summary.html>)
  * [javax.management.monitor](<https://docs.oracle.com/en/java/javase/25/docs/api/java.management/javax/management/monitor/package-summary.html>)
