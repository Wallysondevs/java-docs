# Começando

## 9 Começando

Este capítulo explica o que você precisa fazer para começar com os exemplos de Java Management Extensions (JMX). Ele fornece instruções que se aplicam a todos os exemplos descritos nos capítulos seguintes.

### Informações da Plataforma

Todas as atribuições de variáveis e comandos nos exemplos deste tutorial são definidos usando a sintaxe do UNIX Korn shell. Se você estiver executando um shell diferente do Korn shell em uma plataforma Linux ou macOS, você deve adaptar esses comandos ao seu ambiente de shell preferido.

Se você estiver executando um ambiente operacional Microsoft Windows, na maioria dos casos, a adaptação dos comandos envolverá simplesmente a substituição de barras normais (`/`) por barras invertidas (`\`) e a substituição de dois pontos (`:`) por ponto e vírgula (`;`) nos caminhos. Um comando específico do Microsoft Windows é fornecido apenas quando ele difere significativamente do comando Linux ou macOS fornecido.
