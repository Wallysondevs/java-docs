# Visão Geral da Instalação do JDK

## 1 Visão Geral da Instalação do JDK

O tipo de instalação depende do seu requisito e da plataforma que você escolher para instalar.

Requisitos de Instalação e Tarefas Relacionadas

Para desenvolver ou executar aplicações Java, baixe e instale o JDK.

Formato da String de Versão

Para entender o esquema de string de versão que é usado para distinguir várias versões do JDK, consulte [Formato da String de Versão](<#/doc/guides/install/version-string-format>).

Instalação

O JDK pode ser instalado nas seguintes plataformas:

  * [Microsoft Windows](<#/doc/guides/install/overview-jdk-installation>)

  * [Linux](<#/doc/guides/install/overview-jdk-installation>)

  * [macOS](<#/doc/guides/install/overview-jdk-installation>)

Você pode baixar o JDK na página [Downloads do Java SE Development Kit](<#/>).

Para processadores e navegadores suportados, consulte [Configurações de Sistema Certificadas do Oracle JDK](<https://www.oracle.com/java/technologies/javase/products-doc-jdk25certconfig.html>).

A documentação do JDK é um download separado. Consulte [Documentação do Java SE](<https://www.oracle.com/java/technologies/javase-documentation.html>).

Microsoft Windows

[Instalação do JDK em Plataformas Microsoft Windows](<#/doc/guides/install/installation-jdk-microsoft-windows-platforms>): As instruções incluem etapas para instalar o JDK em sistemas operacionais Microsoft Windows de 64 bits.

Linux

[Instalação do JDK em Plataformas Linux](<#/doc/guides/install/installation-jdk-linux-platforms>): As instruções incluem a instalação a partir de Archive files, Debian Packages, Downloadable RPM Package Files e RPM Packages de Oracle Linux Repositories.

macOS

[Instalação do JDK no macOS](<#/doc/guides/install/installation-jdk-macos>): As instruções incluem etapas para instalar o JDK na plataforma macOS.

Estrutura de Diretórios

Para saber mais sobre os arquivos e diretórios que são criados após a instalação do JDK, consulte [Estrutura de Diretórios Instalados do JDK](<#/doc/guides/install/installed-directory-structure-jdk>).
