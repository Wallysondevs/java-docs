# Monitorar Eventos com a API de Streaming de Eventos do Flight Recorder

## Parte IV Monitorar Eventos com a API de Streaming de Eventos do Flight Recorder

A API de streaming de eventos do Flight Recorder permite que você consuma dados do Flight Recorder continuamente. Esta seção mostra três maneiras de fazer isso:

  * [Criar Stream de Eventos em Processo, Ativo](<#/doc/guides/jrtflightrecorder/create-event-stream-process-active>): Cria um stream de eventos ao mesmo tempo em que uma gravação é criada
  * [Criar Stream de Eventos em Processo, Passivo](<#/doc/guides/jrtflightrecorder/create-event-stream-process-passive>): Cria um stream passivo que escuta por eventos, mas o que é gravado é controlado por meios externos
  * [Criar Stream de Eventos a partir de Processo Externo](<#/doc/guides/jrtflightrecorder/create-event-stream-external-process>): Cria um stream de eventos a partir de um processo Java separado
