# Mudanças Significativas na Versão JDK 11

## Mudanças Significativas na Versão JDK 11

O JDK 11 também teve algumas mudanças significativas. Como o JDK 11 é uma versão de suporte de longo prazo (LTS), você deve estar familiarizado com as seguintes mudanças importantes na versão JDK 11:

  * A Oracle não oferece mais downloads de JRE e Server JRE; consequentemente, o Auto-Update não está mais disponível.

  * Java Web Start, Java Plugin e Java Control Panel não estão disponíveis no JDK. Consulte [Remoção da Pilha de Implantação](<#/doc/guides/migrate/removed-tools-and-components>).

  * O JavaFX não está mais incluído no JDK. Ele agora está disponível como um download separado em <https://openjfx.io/>.

  * JAXB e JAX-WS não são mais empacotados com o JDK. Consulte [Remoção dos Módulos Java EE e CORBA](<#/doc/guides/migrate/removed-tools-and-components>).
