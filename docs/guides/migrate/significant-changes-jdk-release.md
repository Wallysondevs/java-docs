# Mudanças Significativas no JDK

## 2 Mudanças Significativas no JDK
  
Antes de migrar sua aplicação para a versão mais recente do JDK, você deve entender quais são as atualizações e mudanças entre ela e a versão anterior do JDK. Se você estiver migrando do JDK 8, também deve estar familiarizado com as diferenças entre o JDK 8 e as versões posteriores que são descritas em [Migrando do JDK 8 para Versões Posteriores do JDK](<#/doc/guides/migrate/migrating-jdk-8-later-jdk-releases>).

Consulte as seções a seguir para saber mais sobre algumas das mudanças significativas nas versões do JDK.
