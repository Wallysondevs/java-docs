# Mudanças Significativas no Lançamento do JDK 12

## Mudanças Significativas no Lançamento do JDK 12

A seguir estão algumas das adições e atualizações importantes no Java SE 12 e JDK 12:

  * A JVM Constants API foi introduzida para modelar descrições nominais de artefatos chave de arquivos de classe e de tempo de execução, em particular constantes que podiam ser carregadas do pool de constantes. Consulte [JVM Constant API](<https://www.oracle.com/java/technologies/javase/12-relnote-issues.html#JDK-8203252>).
  * A instrução `switch` foi estendida para que possa ser usada tanto como uma instrução quanto como uma expressão. Este é um recurso de linguagem de pré-visualização. Consulte [JEP 325: Switch Expressions (Preview)](<http://openjdk.java.net/jeps/325>) e [JEP 12: Preview Language and VM Features](<http://openjdk.java.net/jeps/12>).
  * Suporte para Unicode 11.0. Consulte [Unicode 11.0](<https://www.oracle.com/java/technologies/javase/12-relnote-issues.html#JDK-8209923>).
  * Foi fornecido suporte a caracteres quadrados para a Era Reiwa japonesa, que começou em maio de 2019. Consulte [Square character support](<https://www.oracle.com/java/technologies/javase/12-relnote-issues.html#JDK-8211398>).
  * O `NumberFormat` adicionou suporte para formatar um número em sua forma compacta. Consulte [Compact Number Formatting Support ](<https://www.oracle.com/java/technologies/javase/12-relnote-issues.html#JDK-8177552>).
