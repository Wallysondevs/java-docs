# Prefácio

## Prefácio

Este guia fornece informações sobre a tecnologia de segurança Java, ferramentas e implementações de algoritmos de segurança, mecanismos e protocolos comumente usados na Plataforma Java, Standard Edition (Java SE).

### Público-alvo

Este documento é destinado a desenvolvedores experientes que constroem aplicações usando o abrangente framework de segurança Java. Ele também é destinado ao usuário ou administrador com um conjunto de ferramentas para gerenciar aplicações de forma segura.
