# Calendários Suportados

## 3 Calendários Suportados

O núcleo da API Date-Time é o pacote [java.time](<https://docs.oracle.com/en/java/javase/25/docs/api/java.base/java/time/package-summary.html>). As classes definidas em java.time baseiam seu sistema de calendário no calendário ISO, que é o padrão mundial para representar data e hora. O calendário ISO segue as regras gregorianas prolépticas. Existem também calendários não-ISO predefinidos no pacote java.time.chrono: os calendários Japonês, Hijrah, Minguo e Budista Tailandês. Para mais informações sobre a API Date-Time, consulte o [Internationalization Trail](<https://docs.oracle.com/javase/tutorial/datetime/iso/index.html>) nos Tutoriais Java.
