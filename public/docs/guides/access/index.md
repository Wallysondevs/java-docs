## Plataforma Java, Edição Padrão

Guia de Acessibilidade Java

Versão 25

G29151-01

Setembro de 2025

Como criar aplicações acessíveis com Java Access Bridge, Java Accessibility API (JAAPI) e Java Accessibility Utilities. Java Access Bridge permite que certas aplicações Java sejam visíveis para tecnologias assistivas no Microsoft Windows. JAAPI permite criar aplicações Java que são acessíveis a pessoas com deficiência. Java Accessibility Utilities é um conjunto de classes utilitárias que ajudam as tecnologias assistivas a fornecer acesso a toolkits GUI que implementam JAAPI.
