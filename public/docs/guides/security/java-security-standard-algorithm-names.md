## Nomes de Algoritmos Padrão

Consulte [Java Security Standard Algorithm Names Specification](<https://docs.oracle.com/en/java/javase/25/docs/specs/security/standard-names.html>) para obter informações sobre o conjunto de nomes padrão para algoritmos, tipos de certificado e de keystore que o Java SE exige e utiliza.
