# Segurança Geral

## 1 Segurança Geral

[Termos e Definições](<#/doc/guides/security/terms-definitions>) lista os termos de criptografia comumente usados e suas definições.

[Visão Geral da Segurança Java](<#/doc/guides/security/java-security-overview1>) fornece uma visão geral da motivação dos principais recursos de segurança, uma introdução às classes de segurança e seu uso, uma discussão sobre o impacto da arquitetura de segurança no código e considerações sobre a escrita de código sensível à segurança.

Certos aspectos da segurança Java podem ser personalizados definindo propriedades de segurança no arquivo de propriedades de segurança. [O Arquivo de Propriedades de Segurança](<#/doc/guides/security/security-properties-file>) descreve este arquivo.

[Especificação de Nomes de Algoritmos Padrão de Segurança Java](<https://docs.oracle.com/en/java/javase/25/docs/specs/security/standard-names.html>) descreve o conjunto de nomes padrão para algoritmos, tipos de certificado e keystore que o Java SE exige e usa.

[Solução de Problemas de Segurança](<#/doc/guides/security/troubleshooting-security>) lista opções para a propriedade de sistema java.security.debug que permitem monitorar o acesso de segurança.
