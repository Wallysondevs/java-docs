# Java Generic Security Services (Java GSS-API)

## 7 Java Generic Security Services (Java GSS-API)

Java Generic Security Services (Java GSS-API) é usado para trocar mensagens de forma segura entre aplicações que se comunicam.

[Introdução aos Tutoriais de JAAS e Java GSS-API](<#/doc/guides/security/introduction-jaas-and-java-gss-api-tutorials1>) é uma série de tutoriais que demonstram vários aspectos do uso do Java Authentication and Authorization Service (JAAS) e do Java GSS-API.

[Acessando GSS-API Nativo](<#/doc/guides/security/accessing-native-gss-api>) descreve como habilitar e configurar o Java GSS-API para usar o GSS-API nativo.

[Programação de Segurança Avançada em Java SE Autenticação, Comunicação Segura e Single Sign-On](<#/doc/guides/security/advanced-security-programming-java-se-authentication-secure-communication-and-single-sign1>) mostra como usar as APIs Java SE GSS para construir aplicações que autenticam seus usuários, se comunicam de forma segura com outras aplicações e serviços, e configurar suas aplicações em um ambiente Kerberos para alcançar o Single Sign-On.

[O Mecanismo Kerberos 5 GSS-API](<#/doc/guides/security/kerberos-5-gss-api-mechanism>) descreve e lista recursos de segurança relacionados ao Java Generic Security Services (Java GSS) para Kerberos 5.

[Solução de Problemas do JGSS](<#/doc/guides/security/troubleshooting-jgss>) descreve como habilitar a depuração no JGSS. Também descreve alguns problemas que podem ocorrer ao tentar um login e sugestões para resolvê-los.
