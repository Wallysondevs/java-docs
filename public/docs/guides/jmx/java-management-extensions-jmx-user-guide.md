# Guia do Usuário da Tecnologia Java Management Extensions

## Parte I Guia do Usuário da Tecnologia Java Management Extensions

Visão Geral da Tecnologia JMX

A tecnologia Java Management Extensions (JMX) foi originalmente desenvolvida através do Java Community Process (JCP) como Java Specification Request (JSR) 3, Java Management Extensions, e JSR 160, JMX Remote API. A JMX API é uma API padrão para gerenciamento e monitoramento de recursos como aplicações, dispositivos, serviços e a Java Virtual Machine.

Usos típicos da tecnologia JMX incluem:

  * Monitoramento e alteração da configuração de aplicações
  * Acumulação de estatísticas sobre o comportamento da aplicação e sua disponibilização
  * Envio de notificações de mudanças de estado e condições errôneas.

A JMX API inclui acesso remoto, de modo que uma aplicação de gerenciamento remota possa interagir com uma aplicação em execução para realizar essas ações.
