# Tutorial da Tecnologia Java Management Extensions (JMX)

## Parte II Tutorial da Tecnologia Java Management Extensions (JMX)

Visão Geral do Tutorial da Tecnologia JMX

Este tutorial fornece exemplos de como usar os principais recursos da tecnologia JMX.

Este tutorial deve ser lido em ordem, do início ao fim, trabalhando com os exemplos à medida que avança. As ações que você executa no início do tutorial podem ser necessárias em partes posteriores do tutorial. Consequentemente, começar no meio do tutorial pode fazer com que você pule ações que são exigidas por certos exemplos.

Onde você deve executar uma tarefa, as instruções são marcadas com um número de ação e as subtarefas são marcadas com uma letra minúscula.

Antes de Usar Este Tutorial

Este tutorial demonstra os conceitos e a tecnologia introduzidos no [Guia do Usuário da Tecnologia Java Management Extensions](<#/doc/guides/jmx/java-management-extensions-jmx-user-guide>). Você deve, portanto, ler a visão geral antes de tentar trabalhar com este tutorial. Para aproveitar ao máximo as informações deste tutorial, você também deve estar familiarizado com os seguintes protocolos e especificações:

  * Remote Method Invocation (RMI)

  * Lightweight Directory Access Protocol (LDAP)

  * Service Location Protocol (SLP)

  * Java Naming and Directory Interface (JNDI) API

  * Java Secure Socket Extension (JSSE)

  * Java Authentication and Authorization Service (JAAS)

  * Java Management Extensions (JMX) Specification, version 1.4

Como Este Tutorial Está Organizado

Este tutorial fornece exemplos nas amplas categorias apresentadas nos seguintes capítulos.

  * [Introdução](<#/doc/guides/jmx/getting-started>) fornece algumas informações de configuração inicial.
  * [Fundamentos da API JMX](<#/doc/guides/jmx/essentials-jmx-api>) apresenta os conceitos centrais da Especificação JMX.
  * [Conectores JMX](<#/doc/guides/jmx/jmx-connectors>) fornece exemplos de como implementar os tipos padrão e dinâmicos de MBean, e realizar operações neles tanto localmente quanto remotamente.
  * [Serviços de Pesquisa](<#/doc/guides/jmx/lookup-services>) demonstra os serviços de pesquisa que podem ser usados em conjunto com a tecnologia JMX.
  * [Segurança](<#/doc/guides/jmx/security>) mostra alguns exemplos de configurações de segurança.
