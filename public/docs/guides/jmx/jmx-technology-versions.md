# Versões da Tecnologia JMX

## 7 Versões da Tecnologia JMX

A Tecnologia Java Management Extensions (JMX) tornou-se uma parte padrão da plataforma Java na Java Platform Standard Edition (Java SE) 5.0. A tecnologia JMX foi desenvolvida através do Java Community Process (JCP) como duas Java Specification Requests (JSRs) intimamente relacionadas. As versões das JSRs implementadas no Java SE 5.0 são detalhadas nas seções a seguir.

### Especificação de Instrumentação e Agente JMX (JSR 3)

A plataforma Java SE 9 e posterior implementa a version 1.4 da JMX Specification (Maintenance Release, October 2006). Ela incorpora as modificações listadas na errata fornecida com o download.

### Especificação da API Remota JMX (JSR 160)

A plataforma Java SE 9 e posterior implementa a version 1.4 da JMX Remote API Specification (Maintenance Release, October 2006).
