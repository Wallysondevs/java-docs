# Exemplos de Java Management Extensions

## Parte III Exemplos de Java Management Extensions

Os arquivos nesta seção são exemplos de código que demonstram algumas das principais funcionalidades da tecnologia JMX. Você pode usar esses exemplos para desenvolver MBeans mais complexos e agentes JMX completos para se adequar à sua solução de gerenciamento.

Cada exemplo consiste em arquivos fonte Java e um arquivo README. O arquivo README explica os tópicos abordados pelo exemplo e as instruções para compilar e executar as classes. Consulte [Tutorial da Tecnologia Java Management Extensions (JMX)](<#/doc/guides/jmx/java-management-extensions-jmx-technology-tutorial>) para descrições mais completas de como executar os exemplos.

JMX Essentials: Apresenta a noção fundamental da API JMX, ou seja, os managed beans, ou MBeans.

JMX MBean Notifications: Implementa notificações de MBean.

MXBeans: Demonstra o uso de MXBeans.

MBean Descriptors: Demonstra o uso de MBean Descriptors.

JMX Connectors: Fornece uma implementação de exemplo de como se conectar a MBeans e realizar operações neles remotamente.

Lookup Services: A API JMX define três vinculações a serviços de lookup, usando tecnologias de lookup existentes.

  * Service Location Protocol (SLP) Lookup Service
  * Java Naming and Directory Interface (JNDI)/LDAP Lookup Service

Security: A API JMX implementa protocolos de segurança existentes para proteger suas conexões.

  * Simple Security
