# Garbage Collection

## 3 Garbage Collection

A HotSpot VM da Oracle inclui vários garbage collectors que você pode usar para ajudar a otimizar o desempenho da sua aplicação. Um garbage collector é especialmente útil se sua aplicação lida com grandes volumes de dados (múltiplos gigabytes), tem muitas threads e possui altas taxas de transação.

Para descrições sobre os garbage collectors disponíveis, consulte [Garbage Collection Implementation](<#/>) no Guia de Otimização de Garbage Collection da Java Platform, Standard Edition HotSpot Virtual Machine.
