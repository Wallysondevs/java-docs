# Fundamentos de Segurança usando Bibliotecas JDK

Esta seção ensina os fundamentos de criptografia e TLS no JDK e apresenta vários exemplos práticos para ilustrar como usar as APIs de segurança.

1. [Introdução à Criptografia/Descriptografia Java](<#/doc/tutorials/security/intro>)

Aprenda como a JCA suporta o trabalho com criptografia em Java e como você pode implementar mecanismos básicos de criptografia/descriptografia usando a Java Security API.

2. [Fundamentos de Assinaturas Digitais e Certificados em Java](<#/doc/tutorials/security/digital-signature>)

Uma introdução sobre como você pode verificar assinaturas digitais e inspecionar certificados dentro do seu código Java.

3. [Monitorando a Segurança de Aplicações Java com ferramentas JDK e Eventos JFR](<#/doc/tutorials/security/monitor>)

Descubra como utilizar keytool, JDK Flight Recorder, JDK Mission Control e JFR Security Events para monitorar a segurança da sua aplicação Java.

4. [Aproveitando Ferramentas e Atualizações do JDK para Ajudar a Proteger Aplicações Java](<#/doc/tutorials/security/app-integrity-tools>)

Aprenda como usar ferramentas e capacidades do JDK para manter as aplicações Java seguras.
