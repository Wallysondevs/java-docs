# As Ferramentas de Monitoramento

Você usa ferramentas e comandos de monitoramento da Java Virtual Machine (JVM) para monitorar e gerenciar aplicações Java e a JVM. As seções a seguir descrevem as ferramentas e comandos do JDK usados para monitorar e gerenciar aplicações Java e a JVM.

1.  [Jfr - Analisar e Imprimir Arquivos do Flight Recorder](<#/doc/tutorials/jvm/tools/monitoring/jfr>)

Analisa e imprime arquivos do Flight Recorder.

2.  [Jconsole - o Monitor Gráfico da Sua Aplicação](<#/doc/tutorials/jvm/tools/monitoring/jconsole>)

Inicia um console gráfico para monitorar e gerenciar aplicações Java.

3.  [Jps - Listando suas JVMs Instrumentadas](<#/doc/tutorials/jvm/tools/monitoring/jps>)

Lista as JVMs instrumentadas no sistema de destino.

4.  [Jstat - Monitorando as Estatísticas das suas JVMs](<#/doc/tutorials/jvm/tools/monitoring/jstat>)

Monitora estatísticas da JVM.

5.  [Jstatd - Monitorando a Criação e Término das suas JVMs](<#/doc/tutorials/jvm/tools/monitoring/jstatd>)

Monitora a criação e o término de JVMs Java HotSpot instrumentadas.
