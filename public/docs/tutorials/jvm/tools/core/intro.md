# Apresentando as Ferramentas Core do JDK

As ferramentas do JDK, e seus comandos, permitem que os desenvolvedores lidem com tarefas de desenvolvimento como compilar e executar um programa, empacotar arquivos-fonte em um arquivo Java Archive (JAR), aplicar políticas de segurança a um arquivo JAR, e muito mais.

O tópico de referência de ferramentas e comandos lista e descreve as ferramentas do Java Development Kit (JDK).

Elas são agrupadas nas seguintes seções com base nas funções relacionadas que executam. Detalhes sobre as ferramentas e os comandos que você usou para executá-las estão contidos nas seções correspondentes deste guia.

## Aviso Legal

Por favor, note que este tutorial não cobre todas as ferramentas do OpenJDK, mas apenas faz uma breve introdução às ferramentas do JDK. Para ler todos os tutoriais relacionados às ferramentas do OpenJDK, você pode encontrá-los [aqui](<https://docs.oracle.com/en/java/javase/26/docs/specs/man/index.html>).
