# A API Stream

A API Stream é sua melhor ferramenta para processar seus dados em memória seguindo uma abordagem map/filter/reduce.

Os tutoriais desta série estão listados abaixo. Eles o guiarão por toda a API Stream, começando pelos conceitos básicos até o design de collectors e streams paralelos.
