## Dominando a API

  * [O Collections Framework](<#/doc/tutorials/api/collections-framework>)

  * [A Stream API](<#/doc/tutorials/api/streams>)

A Java reflection permite que um objeto se olhe no espelho e descubra quais campos, métodos e construtores ele possui. Podemos ler e escrever campos, invocar métodos e até mesmo criar novos objetos chamando os construtores.

  * [A Reflection API](<#/doc/tutorials/reflection>)  


Vamos aprofundar na Reflection API e ver como você pode usá-la para examinar ou modificar o comportamento em tempo de execução de aplicações rodando na Java Virtual Machine.

  * [Virtual Threads](<#/doc/tutorials/new-features/virtual-threads>)

  * [A Foreign Function and Memory API ](<#/doc/tutorials/ffm>)  


A Foreign Function and Memory (FFM) API facilita a interoperabilidade de programas Java com código e dados fora do ambiente de execução Java, sem a fragilidade e o perigo do JNI.
