# Resumo de Operadores

## Operador de Atribuição Simples

Operator | Description
---|---
`=` | Operador de atribuição simples

## Operadores Aritméticos

Operator | Description
---|---
`+` | Operador aditivo (também usado para concatenação de String)
`-` | Operador de subtração
`*` | Operador de multiplicação
`/` | Operador de divisão
`%` | Operador de resto

## Operadores Unários

Operator | Description
---|---
`+` | Operador unário de adição; indica valor positivo (números são positivos sem ele, no entanto)
`-` | Operador unário de subtração; nega uma expressão
`++` | Operador de incremento; incrementa um valor em 1
`--` | Operador de decremento; decrementa um valor em 1
`!` | Operador de complemento lógico; inverte o valor de um boolean

## Operadores de Igualdade e Relacionais

Operator | Description
---|---
`==` | Igual a
`!=` | Diferente de
`>` | Maior que
`>=` | Maior ou igual a
`<` | Menor que
`<=` | Menor ou igual a

## Operadores Condicionais

Operator | Description
---|---
`&&` | AND Condicional
`||` | OR Condicional
`?:` | Ternário (atalho para a instrução `if-then-else`)

## Operador de Comparação de Tipo

Operator | Description
---|---
`instanceof` | Compara um objeto a um tipo especificado

## Operadores Bit a Bit e de Deslocamento de Bits

Operator | Description
---|---
`~` | Complemento bit a bit unário
`<<` | Deslocamento à esquerda com sinal
`>>` | Deslocamento à direita com sinal
`>>>` | Deslocamento à direita sem sinal
`&` | AND bit a bit
`^` | OR exclusivo bit a bit
`|` | OR inclusivo bit a bit
