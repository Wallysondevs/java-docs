# The Foreign Function and Memory API

Estes tutoriais ajudam a compreender as capacidades da API de Função Estrangeira e Memória (FFM API) para invocar código fora da JVM (funções estrangeiras) e para acessar com segurança a memória não gerenciada pela JVM (memória estrangeira). A FFM API facilita que aplicações Java chamem bibliotecas nativas e processem dados nativos sem a fragilidade e o perigo do JNI.

1.  [Acessar Memória Off-Heap ou On-Heap com Segmentos de Memória](<#/doc/tutorials/ffm/access-memory>)

Aprenda como acessar memória off-heap ou on-heap com a API de Função Estrangeira e Memória (FFM API) através de MemorySegment e Arenas.

2.  [Invocar uma Função de Biblioteca C](<#/doc/tutorials/ffm/native>)

Descubra como chamar uma função de biblioteca C a partir do Java.

3.  [Acessar Tipos de Dados Nativos](<#/doc/tutorials/ffm/access-structure>)

Descubra como acessar uma estrutura C com MemoryLayout e fatiar ainda mais segmentos de memória.

4.  [Como Lidar com uma Chamada de Código Nativo de Volta para Código Java](<#/doc/tutorials/ffm/upcall>)

Passe código Java como um ponteiro de função para uma função estrangeira.

5.  [Solucionar Problemas em Chamadas para Funções de Biblioteca Nativa](<#/doc/tutorials/ffm/troubleshoot>)

Invoque funções estrangeiras que retornam ponteiros e verifique possíveis erros.

6.  [Gerar Bindings Java com Jextract](<#/doc/tutorials/ffm/jextract>)

Execute jextract e use os bindings Java gerados para integrar bibliotecas nativas em sua aplicação.
